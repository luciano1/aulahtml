Estrutura Básica

O HTML tem uma estrutura básica que inclui as tags <html>, <head> e <body>.

<html>
    <head>
        <title>Título da Página</title>
    </head>
    <body>
        
    </body>
</html>

Elementos Comuns

  • Cabeçalhos: <h1>, <h2> até <h6>
  • Parágrafos: <p>
  • Listas: <ul> para listas não ordenadas e <ol> para listas ordenadas.
  • Links: <a href="URL">Texto do Link</a>
  • Imagens: <img src="URL" alt="descrição">
  • Tabelas: <table>, <tr>, <td>, <th>
  • Formulários: <form>, <input>, <button>

Tags Semânticas

As tags semânticas melhoram a acessibilidade e a SEO. Exemplos incluem:

  • <header>: Para o cabeçalho da página.
  • <footer>: Para o rodapé.
  • <article>: Para artigos independentes.
  • <section>: Para seções dentro de um documento.
  • <nav>: Para a navegação da página.

Atributos Importantes

  • id: Para identificar um elemento único na página.
  • class: Para agrupar elementos com o mesmo estilo.
  • style: Para adicionar estilo inline.
  • lang: Para definir o idioma do conteúdo.

HTML5

O HTML5 introduziu novas tags e atributos, incluindo:

  • Formulários: Novos tipos de input, como <input type="email"> e <input type="date">.
  • Áudio e Vídeo: <audio> e <video>.
  • Armazenamento Local: localStorage e sessionStorage.
